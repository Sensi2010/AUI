# AUI - Advanced UI | Patchnotes

## Version 3.991

- Fixed an action bar error that could occur when some abilities returned no duration.
- Fixed and synchronized missing localization keys across all language files.
- Completed the remaining translations for Spanish, French, Italian, Japanese, Polish, Russian, and Chinese.
- Updated the default position of the `equipment_status` frame mover element.
- Added gamepad-aware size settings for buffs and debuffs with dynamic sliders.
- Updated the default positions for target buffs and target debuffs to match the current layout.
- Updated the default player debuff position for non-gamepad mode.
- Player debuffs now use the same size in gamepad mode as in non-gamepad mode.
